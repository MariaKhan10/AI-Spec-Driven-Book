# بات چیت کرنے والے روبوٹ

یہ سیکشن **فزیکل AI اور ہومانوئڈ روبوٹکس** کی جدید ایپلیکیشنز اور اہم بنیادی ڈھانچے کے پہلوؤں پر روشنی ڈالتا ہے۔ آپ کی بنیادی اور ایڈوانس AI معلومات کی بنیاد پر، آپ دیکھیں گے کہ ذہین روبوٹس کیسے پیچیدہ **بات چیت کے تعاملات** میں مشغول ہوتے ہیں، بڑے **capstone پروجیکٹس** انجام دیتے ہیں، اور ان کی ترقی اور تعیناتی کے لیے حقیقی دنیا کے ہارڈویئر اور لیبارٹری سیٹ اپ کو سمجھتے ہیں۔  

یہاں پچھلے ماڈیولز کے نظریاتی تصورات عملی نفاذ میں آ جاتے ہیں۔ آپ کو **ملٹی موڈل ہیومن-روبوٹ تعامل**، خود مختار ہومانوئڈ روبوٹ کے انعقاد کے چیلنجز، اور ان جدید سسٹمز کو طاقت دینے والے ضروری ہارڈویئر کی بصیرت حاصل ہوگی۔
